Dataset details: India 100m Population
______________________________________________

DATASET: Version 2.0 estimates for numbers of people per pixel ('ppp') and people per hectare ('pph'), for 2010, 2015 and 2020, with national totals adjusted to match UN population division estimates (http://esa.un.org/wpp/), and remaining unadjusted. Note, an additional dataset is included for the census year, on which 2010, 2015 and 2020 estimates are based.
REGION: Asia
SPATIAL RESOLUTION: 0.000833333 decimal degrees (approx 100m at the equator)
PROJECTION: Geographic WGS84 for 'people per pixel' (ppp) datasets; local projection applied for 'people per hectare' (pph) datasets
UNITS: Estimated persons per grid square 
MAPPING APPROACH: Dasymetric modeling approach incorporating a Random Forest estimation technique (model version 2c), as described in:  Stevens, F. R. et al. (2015). Disaggregating Census Data for Population Mapping Using Random Forests with Remotely-Sensed and Ancillary Data. PLOS ONE, 10(2), e0107042. doi:10.1371/journal.pone.0107043
FORMAT: Geotiff (zipped using 7-zip (open access tool): www.7-zip.org)
FILENAMES: Example  NPL_ppp_2010_adj_v2.tif = Nepal population per pixel (ppp) map for 2010 adjusted to match UN national estimates (adj), dataset version 2 (v2)
DATE OF PRODUCTION: Oct 2017

CITATION: WorldPop. 2017. India 100m Population, Version 2. University of Southampton. DOI: 10.5258/SOTON/WP00532.
